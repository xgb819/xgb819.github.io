# 前期准备

## 域名注册

.com 是全球通用的商业顶级域名，完美适配 B2B 企业交易属性。一个合规、好记的 .com 域名，既能降低海外客户信任成本，又能助力长期 SEO 与品牌传播。

对于企业 B2B 出海独立站，直接用自有品牌名注册对应 .com 域名是最优选择；若品牌 .com 已被抢注，可搭配行业相关词组合，保证域名与品牌强绑定，强化海外客户的品牌认知。同时避免使用连字符，尽量保持域名纯字母、简洁规整，减少流量流失。

在选择域名注册商方面，推荐使用 [哪煮米](https://www.nazhumi.com) 进行域名比价，避开“首年低价、续费暴涨”的套路；Spaceship 作为一家新兴的域名注册商，活动推广的力度很大，网上也可以找到很多公开的优惠码，比如 COMPROS，.com 首年价格只要 20 元出头，续费 70 元，性价比拉满。而且它的中文前端界面做得很出色，交互流畅易懂，不用啃英文界面就能轻松操作。

## 服务器选型与部署

### 服务器选型

| 类型 | 优点 | 缺点 |
| --- | --- | --- |
| 共享主机 | 价格极低，预算友好，稳定性足够起步用 | 资源共享，不适合超大流量、高并发场景 |
| VPS / 轻量应用服务器 | 独立资源不共享，性能稳定 | 价格比共享主机略高 |
| 云服务器 ECS / 独立服务器 | 性能拉满，超高稳定性，企业级合规防护 | 价格昂贵，日常运维成本高，初创期没必要投入 |

对于中小企业而言，B2B 独立站核心是稳运行、低成本获客，海外合规共享主机价格极低，完全满足产品展示、询盘、SEO 基础需求，不用额外投入高成本。

### 服务器配置

服务器 / 主机配置主要由 CPU、内存、带宽、存储空间、流量包等参数组成。共享主机属于多个网站共用一台服务器资源，云服务提供商标注的 CPU、内存、存储空间等参数，大多是理论最高峰值，并非独享固定资源；绝大多数共享主机套餐都会标注流量无限 / 不计量，带宽由整台服务器所有站点共享分配。

相较于 VPS / 轻量应用服务器动辄几百上千的价格，共享主机成本极低，这里同样强推 [Spaceship 网站托管](https://www.spaceship.com/zh/web-hosting)，起步的 Essential 套餐续费价格不到 200 元，并提供 30 天免费试用和 1 年的免费邮箱服务。后期自动备份、企业邮箱等全套服务也可以在 Spaceship 完成购买，不用跨平台管理、减少多平台续费困扰，后台统一管控更省心。

## WordPress 后台配置

完成域名和服务器配置后，直接在域名后加 /wp-admin 即可进入 WordPress 管理员登录面板。使用服务器购买时填写的管理员名称和密码登录。

B2B 出海独立站和 C 端电商站逻辑完全不同，不需要购物车和在线支付等功能，核心围绕「企业展示 + 产品曝光 + 询盘转化 + 安全合规」四大目标，所有配置都围绕这四点展开。

### WordPress 后台基础配置

#### 设置 WordPress 后台操作语言

【操作路径】用户 → 个人资料

下滑找到语言设置区域，在下拉菜单中选择简体中文。

#### 站点基础信息配置

【操作路径】设置 → 常规

- 站点标题：填写企业英文品牌名 + 核心行业关键词

    > 例：ABC Machinery

- 副标题：简短一句话说明企业定位，不超过 20 个单词，突出工厂直营、定制服务等 B2B 核心优势

    > 例：Professional Food Processing Machinery Manufacturer

- 成员资格：默认取消勾选「任何人都可以注册」，B2B 企业站无需客户注册账号，仅保留管理员权限，杜绝恶意注册

- 站点语言：默认 English (United States)，适配海外用户浏览和谷歌搜索引擎识别



- 时区设置：目标欧美市场选 UTC+0/UTC+1，东南亚市场选对应时区，统一时间避免后台时间错乱

##### 关闭默认评论功能

【操作路径】设置 → 讨论

B2B 企业站以询盘为主，可以按需关闭评论，杜绝海外垃圾广告、恶意评论入侵，减少后期站点维护工作量。

##### 固定链接结构配置

【操作路径】设置 → 固定链接

- 固定连接结构：默认文章名

#### 主题安装与配置

【操作路径】外观 → 主题

1. 下滑到页面底部，点击 + 号安装主题，推荐 Astra 主题，加载速度快，内置海量模板，支持自定义页面布局，适配移动端，免费够用；

2. 启用后进入左侧菜单栏的 Astra，点击顶部入门模板菜单，选择 Build with Templates，选择免费模板替换默认主题；

3. 回到 Astra 面板，点击顶部自定义菜单，进入主题自定义面板（面板包含全局、页眉、页脚、网站标识、菜单、小工具、主页设置、额外CSS等核心模块）：

    - 页眉：上传企业 Logo

    - 网站标识：上传网站图标

    - 主页设置：推荐设置为 “静态页面”（而非最新文章）

### 核心页面搭建

#### 添加与编辑页面

多数主题模板安装启用后，会预设好基础页面，仅当预设页面缺失时，才需手动添加页面（【操作路径】页面 → 添加页面）；编辑预设页面或手动添加的页面（【操作路径】页面 → 所有页面），只需要将光标移动到相应页面所在行，点击编辑即可进入编辑页面。

#### 菜单基础配置

【操作路径】外观 → 菜单

- 调整菜单顺序：拖动预设菜单中的选项

- 修改菜单名称：点击菜单选项右侧的下拉箭头，修改“导航标签”，修改后点击“保存菜单”生效

- 添加菜单选项：点击左侧“添加菜单项”选项，勾选页面，点击“添加到菜单”

- 删除菜单选项：点击菜单选项右侧的下拉箭头，点击“移除”按钮，点击“保存菜单”生效

- 绑定菜单位置：在菜单模块底部，可编辑菜单显示位置
